StreamChamps Sports Hub - Pro UI Build

Upload every file/folder in this folder to your GitHub repo.
Then go to Actions > Build StreamChamps APK > Run workflow.
Download the APK artifact and install it on your Pixel.

This UI build adds:
- Dark blue lightning StreamChamps design
- Larger logo hero screen
- League-wide home screen
- NFL, NBA, NHL, Soccer hubs
- Breaking News with story detail pages inside app
- Video News section
- Standings, schedules and player stats source links
- World Cup and soccer league hub
