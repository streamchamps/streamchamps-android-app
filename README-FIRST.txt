STREAMCHAMPS SPORTS HUB - ANDROID APK BUILDER

This project builds an installable Android APK in GitHub Actions.

WHAT TO DO:
1. Create a new PUBLIC GitHub repo named: streamchamps-android-app
2. Upload every file/folder from this zip into that repo.
3. Open the repo on GitHub.
4. Click Actions.
5. Click Build StreamChamps APK.
6. Click Run workflow.
7. Wait for the green check.
8. Open the finished workflow run.
9. Download the artifact named StreamChamps-debug-apk.
10. Unzip it. The file inside is app-debug.apk.
11. Send/open app-debug.apk on your Pixel and install it.

The app includes:
- StreamChamps Sports Hub branding
- NFL news
- NBA news
- NHL news
- Soccer news
- Story detail pages
